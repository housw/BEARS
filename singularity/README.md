
* ## to build the singualrity image file (BlendIt.sif)

```
./build.sh
```

* ## to run the built sif file

```
singularity run BlendIt.sif blendit --help 
# or 
./BlendIt.sif blendit --help 
```

* ## to test
```
singularity test BlendIt.sif
```

* ## to show help
```
singularity run-help BlendIt.sif
```
